### Hugo 目录结构

```
├── archetypes  // 储存.md的模板文件
├── content     // 储存网站的所有内容
├── data        // 储存数据文件供模板调用
├── layouts     // 储存.html模板
├── static      // 储存图片,css,js等静态文件，该目录下的文件会直接拷贝到/public
├── theme       // 储存主题
└── config.toml
```
